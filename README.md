# WorthIt Charge

Mobile-first EV charging companion for WorthIt AI.

The product idea is simple: do not merely kill charging time. Make the charging wait feel worth it.

Current MVP includes:

- Charging session timer from 10 to 60 minutes
- Charge Roulette anonymous nearby-room concept
- Mystery Nearby local discovery mission
- AI Escape Room charging-length story loop
- Charge Odyssey episodic progression
- WorthIt points saved in the browser
- Installable PWA metadata and service worker

This reframes WorthIt AI from only purchase decisions into a broader time-value brand: was this moment worth spending?
